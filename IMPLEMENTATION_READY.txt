================================================================================
                      ❓ NEED HELP?
================================================================================

Problem                          Solution
─────────────────────────────────────────────────────────────────────────────
Backend won't start             → Check Python 3.10+
                                → pip install fastapi uvicorn asyncpg
                                → Port 8000 free? netstat -tuln | grep 8000

Frontend won't start            → Check Node 18+
                                → npm install
                                → Port 3000 free?

Database connection fails       → Check PostgreSQL running
                                → createdb bankchase
                                → python scripts/run_migrations.py

Signup fails                    → Email already exists?
                                → Use different email
                                → psql: DELETE FROM users WHERE email='...'

Login fails                     → Wrong password?
                                → Check email spelling
                                → Try signing up again

Transfer fails                  → Check PIN (last 4 of user ID)
                                → Check balance sufficient
                                → Try again in 1 min if rate limited

More help                       → Read SETUP.md troubleshooting
